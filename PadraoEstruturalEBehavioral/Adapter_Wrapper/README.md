## Adapter / Wrapper

Adapter:

O padrão Adapter é usado quando você precisa fazer com que uma classe existente funcione com outra interface, sem modificar a classe existente. Isso é útil quando você tem uma classe com uma interface que não é compatível com o sistema que deseja usar. O Adapter age como um intermediário entre a classe existente e o sistema que a utiliza, traduzindo as chamadas de método do sistema para chamadas que a classe existente possa entender.

Wrapper:

O padrão Wrapper, também conhecido como Wrapper Pattern, é usado para envolver um objeto para adicionar funcionalidade a ele sem alterar sua estrutura. Ele cria uma nova classe (o "invólucro") que contém uma referência ao objeto original e adiciona ou modifica métodos e comportamentos conforme necessário.
